• infoGetting Started

    Requirements

    • Nodejs 8+

    Run

    1. Download or clone the repository
    2. Using command line
    3. change directory to the unzipped or cloned folder
    4. Run npm install
    5. Copy .env.sample and rename the copy to .env
    • Note: PORT defaults to 8080
    1. Run npm run start
    2. Navigate to localhost:PORT using a browser or call the REST API with curl or similar program.
